## 2. ОБЯЗАТЕЛЬНЫЕ СТРУКТУРНЫЕ ЭЛЕМЕНТЫ ВВЕДЕНИЯ

Все 5 примеров содержат, в этом порядке:

1. **Актуальность** (1-2 параграфа) — почему тема важна, рыночный/практический контекст с цифрами и источниками
2. **Проблема исследования** (1 параграф) — что именно нерешено
3. **Исследовательский вопрос** (явно прописан: «На основании этого, был выдвинут следующий исследовательский вопрос работы: …»)
4. **Цель работы** — «Целью работы является …»
5. **Задачи** — нумерованный список 1)–7)
6. **Объект и предмет** — «Объектом данного исследования являются …; предметом — …» (одно предложение)
7. **Гипотезы H1–HN** — нумерованный список «H1: чем больше X, тем больше/меньше Y, для … компании» (для empirical работ)
8. **Методология** — 1 параграф о методах
9. **Особенность / научная новизна** — что отличает работу от существующих
10. **Практическая значимость** — кому полезны результаты (с примерами: «инвесторы могут …», «менеджеры могут …»)
11. **Структура работы** — «В целях проведения качественного анализа было решено условно разделить исследование на N частей. Первая часть посвящается … Вторая часть …» (повествовательно, не bullets)

**Наш текущий chapter_1** имеет 1, 2, 3, 4, 5, частично 9–10, 11. **Отсутствуют 6 (объект/предмет) и 7 (гипотезы переписать в "чем больше X..." style)**.

## 3. ЯЗЫК — ГЛАВНАЯ ПРОБЛЕМА НАШЕГО ДРАФТА

### Запрещённые конструкции (наши частые ошибки)

| Текущий вариант (плохо) | Целевой вариант (хорошо) |
|---|---|
| post-2011-CBA institutional structure | институциональная структура CBA, действующая с 2011 года |
| Mincer earnings function extension | расширение классической функции заработной платы Минсера |
| year fixed effects | сезонные фиксированные эффекты |
| cluster-robust SE на player_id | стандартные ошибки, кластеризованные на уровне игрока |
| awards signaling channel | канал сигнализации через награды |
| event study вокруг первого All-NBA | событийное исследование относительно первого попадания в All-NBA |
| top5 market premium hypothesis | гипотеза о премии за принадлежность к команде из крупного медиа-рынка |
| Shapley-декомпозиция R² по 9 блокам | декомпозиция объяснённой дисперсии $R^2$ методом значения Шепли по 9 блокам факторов |
| superstar effect | эффект «суперзвёзд» |
| tier dummies | dummy-переменные категорий контракта |
| performance-метрики | показатели игровой результативности |
| Bonferroni и BH-FDR corrections | поправки Бонферрони и Бенджамини—Хохберга |
| robustness checks | проверки робастности |
| structural break | структурный сдвиг |
| supermax-eligibility | право на supermax-контракт |
| anti-marketability finding | антимаркетинговый эффект (дисконт за крупный рынок) |
| contract renewal cycle mechanism | механизм цикла переподписания контрактов |
| informative null | информативный нулевой результат |

### Допустимый код-свитчинг (оставляем)

- Названия лиг, команд: NBA, LAL, BRK, GSW
- Имена: Mincer, Rosen, Hembre, Shapley
- Технические показатели в формулах: PPG, WS, BPM, VORP
- Английские аббревиатуры тестов: RESET, OLS, AIC, BIC, BH-FDR
- CBA — расшифровать при первом упоминании
- Специфические термины контрактов: rookie scale, mid-level, max-25/30/35, supermax (используются как имена tier'ов)

### Правило

При **первом** упоминании англоязычного термина — давать в скобках русский эквивалент. При повторных — можно без объяснения. Текст должен **читаться** на русском, не как калька с английского.

## 4. ОФОРМЛЕНИЕ ТАБЛИЦ И ФИГУР

### Таблицы

Заголовок над таблицей, формат:
```
Таблица 2.1 – Описательные статистики независимых переменных
Источник: расчёты автора
```

Тире (`–`, не дефис `-`), курсив для подзаголовка «Источник:». В Pandoc markdown: `**Таблица 2.1.** Описательные...`

Пример (Dareva):
```
| Переменная | Оценка | P-value | Значимость |
|---|---|---|---|
| Пересечение | 0,65 | <2 e-16 | *** |
| Размер | -0,03 | 0,24 | |
| ... | | | |
```

Под таблицей курсивом — пояснение значимости:
*где * — переменная значима на 5% уровне значимости, ** — на 1%, *** — на 0,1%.*

### Формулы

По центру, с нумерацией справа в скобках:

$$\ln(\text{salary}_{i,t}) = \alpha + \beta_1 \cdot \text{PPG}_{i,t} + \beta_2 \cdot \text{age}_{i,t} + \beta_3 \cdot \text{age}^2_{i,t} + \gamma_t + u_{i,t}$$  (1)

где:
- $\ln(\text{salary}_{i,t})$ — натуральный логарифм реальной зарплаты;
- $\text{PPG}_{i,t}$ — очки за игру;
- ...

### Рисунки

Заголовок ПОД рисунком:
```
Рисунок 4.1 – Декомпозиция объяснённой дисперсии методом Шепли по 9 блокам факторов
Источник: расчёты автора
```

## 5. ЦИТИРОВАНИЕ — APA В ТЕКСТЕ (как в Tyulenev 2025)

**Стиль:**
- Один автор: `(Минсер, 1974)` или `Минсер (1974) показал, что …`
- Два автора: `(Lazear & Rosen, 1981)` — амперсанд в скобках, «и» в тексте: `Lazear и Rosen (1981) разработали…`
- Три и более: `(Cameron et al., 2011)`
- Группа: `(Hembre, 2022; Kleven et al., 2013)`

**Сноски** в Dareva — full bibliographic reference в footnote (GOST-style). У нас APA — это **другой стиль**, и наш стиль легитимен (HSE guideline §6.3.1). Не смешиваем.

**Список литературы** — алфавитный, без нумерации, по APA-7. Уже сделан в `analysis_v2/reports/references_apa.md`.

## 6. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ HSE (§6.1 гайдлайна)

| Параметр | Значение |
|---|---|
| Формат бумаги | A4 |
| Поле слева | 35 мм |
| Поле справа | ≥10 мм |
| Поле сверху/снизу | ≥20 мм |
| Шрифт | Times New Roman |
| Основной текст | 12 пт, межстрочный 1,5, justified |
| Список литературы | 12 пт, межстрочный 1,5 |
| Таблицы/фигуры | 10–12 пт, межстрочный 1,0 |
| Сноски | 10 пт, межстрочный 1,0 |
| Заголовки | centered |
| Отступ первой строки | 1,25 см |
| Цвет | чёрный (графики могут быть цветными) |
| Bold/italics | по одному за раз, **не вместе** |
| Underlined | **запрещено** |
| Нумерация страниц | внизу, с 2-й страницы, без точки |
| Каждый раздел | с новой страницы |
